GitHub használatának alapvető parancsai

Helyi (local) Git parancsok

  • git init új Git repó inicializálása
  • git status módosítások állapotának megtekintése
  • git add . összes módosított fájl hozzáadása a staging területhez
  • git commit -m "üzenet" változások mentése egy commitban
  • git log commit előzmények megtekintése

Távoli (remote) GitHub parancsok

  • git remote add origin [url] GitHub távoli repo hozzáadása
  • git push -u origin main első feltöltés a GitHub-ra
  • git push további módosítások feltöltése
  • git pull legfrissebb módosítások letöltése a GitHub-ról
  • git clone [url] meglévő projekt letöltése GitHub-ról

Hasznos VS Code bővítmények

  • GitLens fejlett Git-előzmények megjelenítése
  • GitHub Pull Requests and Issues GitHub integráció VS Code-ban
  • Git Graph vizuális commit fa
  • GitHub Copilot mesterséges intelligencia alapú kódkiegészítés
